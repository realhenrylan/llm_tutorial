❌ 常见错误有哪些？

通俗理解：编程就像开车，总会遇到各种问题。了解常见错误可以让你更快地找到问题所在。

常见类型：维度不匹配、设备不一致、梯度为零、数据类型错误

🎨 可视化演示：常见错误类型

📐
维度错误
形状不匹配
💻
设备错误
CPU/GPU不匹配
📊
梯度错误
梯度为零/NaN
📝
类型错误
数据类型不兼容

遇到错误先看报错信息，通常会告诉你具体是哪里出了问题！

📝 代码示例：常见错误及解决方法

# 常见错误及解决方法
import torch
import torch.nn as nn

# 错误1：维度不匹配
print("❌ 错误1：维度不匹配")
model = nn.Linear(10, 1)
try:
    x = torch.randn(32, 20)  # 输入维度是20，但模型期望10
    output = model(x)
except Exception as e:
    print("  错误信息:", str(e)[:80])
    print("  ✅ 解决方法: 确保输入维度与模型匹配")

# 错误2：设备不一致
print("\n❌ 错误2：设备不一致")
model = nn.Linear(10, 1).to('cuda') # 模型在GPU上
try:
    x = torch.randn(32, 10)  # 数据在CPU上
    output = model(x)
except Exception as e:
    print("  错误信息:", str(e)[:80])
    print("  ✅ 解决方法: 使用x.to('cuda')或model.to('cpu')")

# 错误3：梯度为零
print("\n❌ 错误3：梯度为零")
x = torch.randn(10, requires_grad=True)
y = x.detach()  # 断开梯度追踪
y.backward()
print("  x.grad:", x.grad)
print("  ✅ 解决方法: 检查是否意外调用了detach()")

# 错误4：数据类型错误
print("\n❌ 错误4：数据类型错误")
model = nn.Linear(10, 1)
try:
    x = torch.randn(32, 10).int()  # 整型数据
    output = model(x)
except Exception as e:
    print("  错误信息:", str(e)[:80])
    print("  ✅ 解决方法: 使用x.float()转换为浮点型")

🛠️ 常用调试工具有哪些？

通俗理解：调试工具就像医生的听诊器，可以帮助你诊断模型的"健康状况"。

常用工具：print、pdb、torchinfo、tensorboard、gradcheck

🎨 可视化演示：调试工具

🛠️ 调试工具一览
📝 print: 最简单直接，打印张量形状和值
🔍 pdb: Python调试器，设置断点
📊 torchinfo: 打印模型结构和参数
📈 tensorboard: 可视化训练过程
🔧 gradcheck: 检查梯度计算是否正确

先用print定位问题，复杂问题用pdb，模型结构用torchinfo！

📝 代码示例：调试工具

# 调试工具示例
import torch
import torch.nn as nn

# 1. 使用print调试
print("🛠️ 调试工具1: print")
x = torch.randn(32, 10)
print("  x.shape:", x.shape)
print("  x.dtype:", x.dtype)
print("  x.device:", x.device)
print("  x.requires_grad:", x.requires_grad)

# 2. 使用torchinfo查看模型
print("\n🛠️ 调试工具2: torchinfo")
try:
    from torchinfo import summary
    model = nn.Sequential(
        nn.Linear(10, 32),
        nn.ReLU(),
        nn.Linear(32, 1)
    )
    print("  模型摘要已输出")
except:
    print("  需要安装: pip install torchinfo")

# 3. 使用gradcheck检查梯度
print("\n🛠️ 调试工具3: gradcheck")
def my_func(x):
    return x * x + 2 * x

x = torch.randn(10, requires_grad=True)
from torch.autograd import gradcheck
test = gradcheck(my_func, x, eps=1e-6, atol=1e-4)
print("  gradcheck通过:", test)

# 4. 检查NaN和Inf
print("\n🛠️ 调试工具4: 检查NaN/Inf")
y = torch.tensor([1.0, float('nan'), float('inf'), 2.0])
print("  has_nan:", torch.isnan(y).any())
print("  has_inf:", torch.isinf(y).any())
print("  清理后:", torch.nan_to_num(y))

✅ 调试检查清单

通俗理解：检查清单就像登机前的安检，确保所有东西都准备好了。每次训练前都应该检查一遍。

检查项目：数据加载、模型结构、损失函数、优化器、训练循环

🎨 可视化演示：调试检查清单

✅ 调试检查清单
📦 数据检查: 形状、类型、设备、范围
🧠 模型检查: 结构、参数数量、初始化
📉 损失检查: 类型、输入格式、数值范围
🚀 优化器检查: 参数、学习率、权重衰减
🔄 训练检查: train/eval模式、梯度清零、设备一致性
📊 验证检查: 验证集划分、评估指标、早停条件

遇到问题时，按照清单逐项检查，总能找到问题所在！

📝 代码示例：调试检查函数

# 调试检查函数
import torch
import torch.nn as nn

def debug_checklist(model, dataloader, criterion, optimizer, device):
    print("✅ 调试检查清单")
    print("-" * 50)
    
    # 1. 数据检查
    print("\n📦 1. 数据检查")
    inputs, targets = next(iter(dataloader))
    print(f"   输入形状: {inputs.shape}")
    print(f"   目标形状: {targets.shape}")
    print(f"   数据类型: {inputs.dtype}")
    print(f"   数据设备: {inputs.device}")
    
    # 2. 模型检查
    print("\n🧠 2. 模型检查")
    total_params = sum(p.numel() for p in model.parameters())
    trainable_params = sum(p.numel() for p in model.parameters() if p.requires_grad)
    print(f"   总参数: {total_params}")
    print(f"   可训练参数: {trainable_params}")
    print(f"   模型设备: {next(model.parameters()).device}")
    
    # 3. 前向传播测试
    print("\n🔄 3. 前向传播测试")
    model.eval()
    with torch.no_grad():
        inputs_device = inputs.to(device)
        outputs = model(inputs_device)
        print(f"   输出形状: {outputs.shape}")
        print(f"   输出范围: [{outputs.min():.4f}, {outputs.max():.4f}]")
        print(f"   无NaN: {not torch.isnan(outputs).any()}")
    
    # 4. 损失计算测试
    print("\n📉 4. 损失计算测试")
    targets_device = targets.to(device)
    loss = criterion(outputs, targets_device)
    print(f"   损失值: {loss.item():.4f}")
    print(f"   损失非负: {loss.item() >= 0}")
    
    # 5. 反向传播测试
    print("\n🔙 5. 反向传播测试")
    model.train()
    optimizer.zero_grad()
    outputs = model(inputs_device)
    loss = criterion(outputs, targets_device)
    loss.backward()
    
    grad_norms = [p.grad.norm().item() for p in model.parameters() if p.grad is not None]
    print(f"   梯度范数: [{min(grad_norms):.4e}, {max(grad_norms):.4e}]")
    print(f"   无NaN梯度: {all(not torch.isnan(p.grad).any() for p in model.parameters() if p.grad is not None)})
    
    print("\n✅ 检查完成！")

print("调试检查函数已定义")

✏️ 调试实战练习

通俗理解：现在我们来实战调试一个有问题的模型，找出并修复所有问题。

在LLM中的作用：调试是开发过程中必不可少的技能，特别是在训练大型模型时。

📝 完整示例：调试有问题的模型

# 调试有问题的模型
import torch
import torch.nn as nn
import torch.optim as optim

class BuggyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(10, 32)
        self.fc2 = nn.Linear(16, 1)  # Bug1: 维度不匹配
    
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# 调试过程
print("🔍 开始调试...")

model = BuggyModel()
x = torch.randn(32, 10)

try:
    output = model(x)
    print("✅ 前向传播成功")
except Exception as e:
    print(f"❌ 错误: {e}")
    print("💡 提示: 检查fc1输出维度(32)和fc2输入维度(16)")
    
    # 修复Bug
    print("\n🔧 修复中...")
    model.fc2 = nn.Linear(32, 1)
    output = model(x)
    print("✅ 修复成功！")
    print(f"输出形状: {output.shape}")
