🔹 什么是卷积？

通俗理解：卷积就像用一个小滤镜在图片上滑动，提取图片的特征，比如边缘、纹理等。

在LLM中的作用：虽然LLM主要用Transformer，但CNN在图像理解和多模态模型中仍然很重要。

📝 代码示例

# 卷积操作
import torch
import torch.nn as nn

# 创建卷积层
# in_channels: 输入通道数（RGB=3）
# out_channels: 输出通道数（卷积核数量）
# kernel_size: 卷积核大小
# stride: 步长
# padding: 填充
conv = nn.Conv2d(in_channels=3, out_channels=16, kernel_size=3, stride=1, padding=1)

# 模拟输入图像 (batch, channels, height, width)
img = torch.randn(2, 3, 32, 32)

# 前向传播
output = conv(img)
print("输入形状：", img.shape)
print("输出形状：", output.shape)

📏 什么是池化？

通俗理解：池化就像把图片缩小，只保留最重要的信息，比如最大值或平均值。

在LLM中的作用：减少特征图的尺寸，降低计算量，增加感受野。

Max Pooling取窗口内最大值，Average Pooling取窗口内平均值！

📝 代码示例

# 池化操作
import torch
import torch.nn as nn

# Max Pooling
max_pool = nn.MaxPool2d(kernel_size=2, stride=2)

# Average Pooling
avg_pool = nn.AvgPool2d(kernel_size=2, stride=2)

# Adaptive Average Pooling（输出固定尺寸）
adaptive_pool = nn.AdaptiveAvgPool2d(output_size=(7, 7))

# 模拟输入
x = torch.randn(2, 16, 32, 32)

print("输入形状：", x.shape)
print("Max Pool输出：", max_pool(x).shape)
print("Avg Pool输出：", avg_pool(x).shape)
print("Adaptive Pool输出：", adaptive_pool(x).shape)

🏗️ 什么是CNN架构？

通俗理解：CNN架构就像一个流水线，先提取低级特征（边缘），再提取中级特征（纹理），最后提取高级特征（物体）。

在LLM中的作用：用于处理图像输入，是多模态模型的重要组成部分。

🎨 可视化演示：CNN架构示意图

🏗️ 经典CNN架构
📥 输入层: (batch, 3, 224, 224) — RGB图像
🔹 Conv1: 提取边缘特征
📏 Pool1: 降采样
🔹 Conv2: 提取纹理特征
📏 Pool2: 降采样
🔹 Conv3: 提取高级特征
📏 Pool3: 降采样
🧠 FC层: 分类决策
📤 输出层: (batch, 1000) — 分类概率

卷积层负责提取特征，池化层负责降采样，全连接层负责分类！

📝 代码示例

# 完整CNN架构
import torch
import torch.nn as nn

class SimpleCNN(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()
        self.features = nn.Sequential(
            nn.Conv2d(3, 32, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, stride=2),
            
            nn.Conv2d(32, 64, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, stride=2),
            
            nn.Conv2d(64, 128, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, stride=2)
        )
        
        self.classifier = nn.Sequential(
            nn.Linear(128 * 4 * 4, 512),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(512, num_classes)
        )

    def forward(self, x):
        x = self.features(x)
        x = x.view(-1, 128 * 4 * 4)
        x = self.classifier(x)
        return x

model = SimpleCNN()
print(model)

✏️ CNN实战练习

通俗理解：现在我们来实际训练一个CNN模型，看看它在MNIST数据集上的表现。

在LLM中的作用：理解CNN的训练流程，为后续学习多模态模型打下基础。

📝 完整训练示例

# CNN训练完整示例
import torch
import torch.nn as nn
import torch.optim as optim

class MNISTCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv1 = nn.Conv2d(1, 32, 3, 1)
        self.conv2 = nn.Conv2d(32, 64, 3, 1)
        self.fc1 = nn.Linear(9216, 128)
        self.fc2 = nn.Linear(128, 10)

    def forward(self, x):
        x = torch.relu(self.conv1(x))
        x = torch.max_pool2d(x, 2)
        x = torch.relu(self.conv2(x))
        x = torch.max_pool2d(x, 2)
        x = torch.flatten(x, 1)
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# 创建模型、损失函数和优化器
model = MNISTCNN()
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=1e-3)

# 模拟训练
model.train()
for epoch in range(3):
    optimizer.zero_grad()
    x = torch.randn(32, 1, 28, 28)
    y = torch.randint(0, 10, (32,))
    output = model(x)
    loss = criterion(output, y)
    loss.backward()
    optimizer.step()
    print(f"Epoch {epoch}, Loss: {loss.item():.4f}")
