📈 什么是评估指标？

通俗理解：评估指标就像考试分数，用来衡量模型学得好不好。不同的任务需要不同的指标。

分类任务：准确率、精确率、召回率、F1分数

回归任务：MSE、MAE、R²分数

🎨 可视化演示：评估指标

✅
准确率
正确预测比例
🔍
精确率
预测正确的比例
🎯
召回率
实际正确被找到的比例
⚖️
F1分数
精确率和召回率的调和平均

不平衡数据时，准确率不可靠，需要看精确率和召回率！

📝 代码示例：评估指标

# 评估指标
import torch
import torch.nn as nn

# 模拟预测和真实值
predictions = torch.tensor([0, 1, 1, 0, 1, 0, 1, 0])
targets = torch.tensor([0, 1, 0, 0, 1, 1, 1, 0])

# 1. 准确率
accuracy = (predictions == targets).float().mean()
print("准确率:", accuracy.item())

# 2. 精确率（Precision）
tp = ((predictions == 1) & (targets == 1)).sum().float()
fp = ((predictions == 1) & (targets == 0)).sum().float()
precision = tp / (tp + fp)
print("精确率:", precision.item())

# 3. 召回率（Recall）
fn = ((predictions == 0) & (targets == 1)).sum().float()
recall = tp / (tp + fn)
print("召回率:", recall.item())

# 4. F1分数
f1 = 2 * precision * recall / (precision + recall)
print("F1分数:", f1.item())

# 5. 回归指标
reg_pred = torch.tensor([2.5, 3.2, 1.8])
reg_target = torch.tensor([3.0, 3.0, 2.0])

mse = nn.MSELoss()(reg_pred, reg_target)
mae = nn.L1Loss()(reg_pred, reg_target)
r2 = 1 - (mse / reg_target.var())

print("\n回归指标:")
print("MSE:", mse.item())
print("MAE:", mae.item())
print("R²:", r2.item())

🧮 什么是混淆矩阵？

通俗理解：混淆矩阵就像一张成绩单，告诉你模型在每个类别上的表现——哪些预测对了，哪些预测错了。

四个指标：真正例(TP)、假正例(FP)、假反例(FN)、真反例(TN)

🎨 可视化演示：混淆矩阵

🧮 混淆矩阵结构
✅ TP（真正例）: 预测为正，实际为正
❌ FP（假正例）: 预测为正，实际为负
❌ FN（假反例）: 预测为负，实际为正
✅ TN（真反例）: 预测为负，实际为负

对角线是正确预测，非对角线是错误预测！

📝 代码示例：混淆矩阵

# 混淆矩阵
import torch

# 计算混淆矩阵
def confusion_matrix(predictions, targets, num_classes=2):
    matrix = torch.zeros(num_classes, num_classes)
    for pred, target in zip(predictions, targets):
        matrix[target][pred] += 1
    return matrix

# 二分类示例
predictions = torch.tensor([0, 1, 1, 0, 1, 0, 1, 0])
targets = torch.tensor([0, 1, 0, 0, 1, 1, 1, 0])

cm = confusion_matrix(predictions, targets)
print("二分类混淆矩阵:")
print(cm)
print("\n解释:")
print("TP =", cm[1, 1].item())
print("FP =", cm[0, 1].item())
print("FN =", cm[1, 0].item())
print("TN =", cm[0, 0].item())

# 多分类示例
multi_pred = torch.tensor([0, 1, 2, 0, 1, 2, 0, 1, 2])
multi_target = torch.tensor([0, 1, 2, 0, 0, 2, 1, 1, 1])

multi_cm = confusion_matrix(multi_pred, multi_target, num_classes=3)
print("\n多分类混淆矩阵:")
print(multi_cm)

🔄 什么是交叉验证？

通俗理解：交叉验证就像多次考试，每次用不同的题目，确保模型稳定可靠。避免一次考试的偶然因素。

常用方法：K折交叉验证、留一法、分层K折

🎨 可视化演示：K折交叉验证

🔄 K折交叉验证流程
📦 步骤1: 将数据分成K个折（比如K=5）
🔄 步骤2: 轮流用1折做验证，其余K-1折做训练
📊 步骤3: 取K次结果的平均值作为最终评估
🎯 优点: 充分利用数据，评估更可靠

K值常用5或10，数据少时K可以大一些！

📝 代码示例：交叉验证

# K折交叉验证
import torch
import torch.nn as nn
import torch.optim as optim

class SimpleModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc = nn.Linear(10, 2)
    
    def forward(self, x):
        return self.fc(x)

# K折交叉验证实现
def k_fold_cross_validation(X, y, k=5):
    fold_size = len(X) // k
    accuracies = []
    
    for i in range(k):
        # 划分训练集和验证集
        val_start = i * fold_size
        val_end = val_start + fold_size
        
        X_train = torch.cat([X[:val_start], X[val_end:]])
        y_train = torch.cat([y[:val_start], y[val_end:]])
        X_val = X[val_start:val_end]
        y_val = y[val_start:val_end]
        
        # 训练模型
        model = SimpleModel()
        criterion = nn.CrossEntropyLoss()
        optimizer = optim.Adam(model.parameters(), lr=0.001)
        
        for epoch in range(50):
            optimizer.zero_grad()
            output = model(X_train)
            loss = criterion(output, y_train)
            loss.backward()
            optimizer.step()
        
        # 评估
        model.eval()
        with torch.no_grad():
            val_output = model(X_val)
            accuracy = (val_output.argmax(dim=1) == y_val).float().mean().item()
            accuracies.append(accuracy)
        
        print(f"折{i+1}: 准确率={accuracy:.4f}")
    
    print(f"\n平均准确率={torch.tensor(accuracies).mean().item():.4f}")
    print(f"标准差={torch.tensor(accuracies).std().item():.4f}")
    return accuracies

# 模拟数据
X = torch.randn(100, 10)
y = torch.randint(0, 2, (100,))

# 执行交叉验证
accuracies = k_fold_cross_validation(X, y, k=5)

✏️ 模型评估实战练习

通俗理解：现在我们来综合运用各种评估方法，对模型进行全面评估。

在LLM中的作用：模型评估是确保模型质量的关键，特别是在部署前需要全面测试。

📝 完整示例：模型评估流程

# 完整模型评估流程
import torch
import torch.nn as nn

class Evaluator:
    def __init__(self, model, device):
        self.model = model.to(device)
        self.device = device
    
    def evaluate(self, dataloader):
        self.model.eval()
        all_preds = []
        all_targets = []
        
        with torch.no_grad():
            for inputs, targets in dataloader:
                inputs, targets = inputs.to(self.device), targets.to(self.device)
                outputs = self.model(inputs)
                all_preds.extend(outputs.argmax(dim=1).cpu().tolist())
                all_targets.extend(targets.cpu().tolist())
        
        all_preds = torch.tensor(all_preds)
        all_targets = torch.tensor(all_targets)
        
        accuracy = (all_preds == all_targets).float().mean().item()
        print(f"准确率: {accuracy:.4f}")
        return accuracy

print("✅ 评估器已准备就绪")
print("评估流程: 加载模型 → 前向传播 → 计算指标 → 输出结果")

📊 模型评估方法对比表

方法类型作用优点适用场景
准确率指标正确预测比例简单直观均衡数据
精确率指标预测为正的正确率关注假阳性不平衡数据
召回率指标实际为正的覆盖率关注假阴性不平衡数据
F1分数指标精确率和召回率调和平均综合评估不平衡数据
混淆矩阵工具详细分类结果全面分析多分类
K折交叉验证方法多次训练验证评估可靠数据量小
