🔍 什么是注意力机制？

通俗理解：注意力机制就像我们看书时，会重点关注某些关键字，忽略不重要的内容。模型会根据重要程度分配不同的权重。

在LLM中的作用：让模型能够关注输入中最重要的信息，是Transformer的核心。

🎨 可视化演示：注意力机制

🔍 Attention机制公式
Attention(Q, K, V) = softmax(QKᵀ/√dₖ)V
❓ Q (Query): 当前问题/查询
🔑 K (Key): 键，用于匹配
📦 V (Value): 值，实际内容

注意力机制通过计算Q和K的相似度，来决定从V中提取多少信息！

📝 代码示例

# 注意力机制
import torch
import torch.nn as nn

# 缩放点积注意力
def scaled_dot_product_attention(Q, K, V, mask=None):
    d_k = Q.size(-1)
    scores = torch.matmul(Q, K.transpose(-2, -1)) / torch.sqrt(torch.tensor(d_k, dtype=torch.float32))
    
    if mask is not None:
        scores = scores.masked_fill(mask == 0, -1e9)
    
    attn_weights = torch.softmax(scores, dim=-1)
    output = torch.matmul(attn_weights, V)
    return output, attn_weights

# 多头注意力
multihead_attn = nn.MultiheadAttention(embed_dim=512, num_heads=8, batch_first=True)

# 模拟输入
x = torch.randn(32, 10, 512)

# 前向传播
output, attn_weights = multihead_attn(x, x, x)
print("输出形状：", output.shape)
print("注意力权重形状：", attn_weights.shape)

🧠 什么是Encoder？

通俗理解：Encoder就像一个翻译机的"理解"部分，它读取输入的句子，将其转换为一种内部表示。

在LLM中的作用：编码输入序列，提取关键信息，为后续的解码做准备。

🎨 可视化演示：Encoder结构

🧠 Encoder层结构
🔍 Multi-Head Attention: 多头自注意力
🔄 Add & Norm: 残差连接 + 层归一化
⚡ Feed Forward: 前馈神经网络
🔄 Add & Norm: 残差连接 + 层归一化

Encoder通常堆叠多层，每层都包含注意力和前馈网络！

📝 什么是Decoder？

通俗理解：Decoder就像一个翻译机的"生成"部分，它根据Encoder的理解，一步步生成输出序列。

在LLM中的作用：生成文本，每次生成一个token，直到遇到结束符。

🎨 可视化演示：Decoder结构

📝 Decoder层结构
🔍 Masked Multi-Head Attention: 带掩码的自注意力
🔄 Add & Norm: 残差连接 + 层归一化
🔍 Cross Attention: 编码器-解码器注意力
🔄 Add & Norm: 残差连接 + 层归一化
⚡ Feed Forward: 前馈神经网络
🔄 Add & Norm: 残差连接 + 层归一化

Decoder有两个注意力层：自注意力和交叉注意力！

✏️ Transformer实战练习

通俗理解：现在我们来构建一个完整的Transformer模型进行序列到序列的任务。

在LLM中的作用：理解完整的Transformer架构，为学习GPT、BERT等模型打下基础。

📝 完整示例：Transformer模型

# 完整Transformer模型
import torch
import torch.nn as nn

class TransformerModel(nn.Module):
    def __init__(self, vocab_size, d_model=512, nhead=8, num_layers=6):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.transformer = nn.Transformer(
            d_model=d_model, nhead=nhead, num_encoder_layers=num_layers,
            num_decoder_layers=num_layers, batch_first=True
        )
        self.fc = nn.Linear(d_model, vocab_size)

    def forward(self, src, tgt):
        src = self.embedding(src)
        tgt = self.embedding(tgt)
        output = self.transformer(src, tgt)
        return self.fc(output)

# 创建模型
model = TransformerModel(vocab_size=10000)

# 模拟输入
src = torch.randint(0, 10000, (32, 10))
tgt = torch.randint(0, 10000, (32, 8))

# 前向传播
output = model(src, tgt)

print("输入形状：", src.shape, tgt.shape)
print("输出形状：", output.shape)
