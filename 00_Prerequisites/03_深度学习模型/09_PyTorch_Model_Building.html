🧩 什么是模型结构？

通俗理解：模型结构就像一个流程图，定义了数据从输入到输出要经过哪些处理步骤。

在LLM中的作用：决定了模型的能力和性能，好的结构能让模型更好地学习数据规律。

🎨 可视化演示：模型结构示意图

🏗️ 典型神经网络结构
📥 Input (batch, 784)
Linear(784 → 256)
ReLU()
Linear(256 → 128)
ReLU()
Linear(128 → 10)
📤 Output (batch, 10)

模型就像一个流水线，数据从输入流入，经过层层处理，最后输出结果！

📝 代码示例

# 定义简单模型结构
import torch
import torch.nn as nn

class MLP(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super().__init__()
        self.layers = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, output_dim)
        )

    def forward(self, x):
        return self.layers(x)

model = MLP(input_dim=784, hidden_dim=256, output_dim=10)
print(model)

📦 常用层有哪些？

通俗理解：常用层就像不同的工具，每种层都有自己的作用，比如线性层做变换，激活函数增加非线性。

在LLM中的作用：构建复杂的神经网络，让模型能够学习复杂的模式。

🎨 可视化演示：常用层对比

📦 常用层介绍
Linear — 线性变换 y = Wx + b
ReLU — 非线性激活 max(0, x)
Conv2d — 卷积操作
MaxPool2d — 池化降采样
Dropout — 正则化随机失活
BatchNorm — 批量归一化

Linear层是最基础的层，几乎所有模型都会用到！

📝 代码示例

# 常用层示例
import torch
import torch.nn as nn

# 线性层
linear = nn.Linear(10, 5)
x = torch.randn(3, 10)
print("Linear输出：", linear(x).shape)

# 卷积层
conv = nn.Conv2d(in_channels=3, out_channels=16, kernel_size=3)
img = torch.randn(2, 3, 32, 32)
print("Conv输出：", conv(img).shape)

# 池化层
pool = nn.MaxPool2d(kernel_size=2)
print("Pool输出：", pool(conv(img)).shape)

# Dropout
dropout = nn.Dropout(p=0.5)
print("Dropout输出：", dropout(x).shape)

🔗 什么是模块化设计？

通俗理解：模块化设计就像搭积木，把复杂的模型拆分成小模块，每个模块负责一部分功能。

在LLM中的作用：让代码更清晰、更易维护，方便复用和扩展。

🎨 可视化演示：模块化设计

🔗 模块化设计示意图
• Encoder模块 — 编码输入信息
• Decoder模块 — 生成输出
• Attention模块 — 关注重要信息
• FeedForward模块 — 非线性变换

每个模块可以独立测试和优化，就像乐高积木一样！

📝 代码示例

# 模块化设计示例
import torch
import torch.nn as nn

class AttentionBlock(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.query = nn.Linear(dim, dim)
        self.key = nn.Linear(dim, dim)
        self.value = nn.Linear(dim, dim)

    def forward(self, x):
        q = self.query(x)
        k = self.key(x)
        v = self.value(x)
        return q + k + v

class TransformerLayer(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.attn = AttentionBlock(dim)
        self.ffn = nn.Sequential(
            nn.Linear(dim, dim * 4),
            nn.ReLU(),
            nn.Linear(dim * 4, dim)
        )

    def forward(self, x):
        x = x + self.attn(x)
        x = x + self.ffn(x)
        return x

model = TransformerLayer(dim=64)
print(model)

⚙️ 参数初始化是什么？

通俗理解：参数初始化就像给模型的"大脑"设置初始状态，好的初始化能让模型更快收敛。

在LLM中的作用：确保训练稳定进行，避免梯度消失或爆炸。

🎨 可视化演示：常用初始化方法

⚙️ 参数初始化方法
Xavier — 适用于tanh/sigmoid
He — 适用于ReLU
Normal — 标准正态分布
Uniform — 均匀分布

ReLU激活函数推荐使用He初始化，tanh推荐使用Xavier初始化！

📝 代码示例

# 参数初始化
import torch
import torch.nn as nn

class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(10, 20)
        self.fc2 = nn.Linear(20, 5)
        self._init_weights()

    def _init_weights(self):
        for m in self.modules():
            if isinstance(m, nn.Linear):
                nn.init.kaiming_normal_(m.weight, mode='fan_in', nonlinearity='relu')
                if m.bias is not None:
                    nn.init.zeros_(m.bias)

    def forward(self, x):
        return self.fc2(torch.relu(self.fc1(x)))

model = MyModel()
print("fc1权重均值：", model.fc1.weight.data.mean())
print("fc1偏置：", model.fc1.bias.data)
