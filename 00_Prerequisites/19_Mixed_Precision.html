📚 什么是混合精度训练？

通俗理解：混合精度训练就像用不同精度的尺子来测量东西。大部分计算用FP16（短尺子），关键部分用FP32（长尺子）。这样既快又准。

为什么需要？FP16计算速度快、内存占用少，但精度低；FP32精度高但速度慢。混合精度结合两者优点。

🔢 FP16训练

通俗理解：FP16训练就像用一辆跑得快的赛车，但需要一个好的刹车系统（梯度缩放）来防止翻车（梯度下溢）。

关键组件：torch.autocast自动选择精度、GradScaler缩放梯度、AMP自动混合精度

🎨 可视化演示：FP16训练流程

🔢 FP16训练流程
📥 输入数据: FP32格式
⚡ 前向传播: autocast自动转为FP16
📉 损失计算: 自动混合精度
🔧 梯度缩放: GradScaler放大梯度
🔙 反向传播: FP16计算梯度
📐 参数更新: 缩放回FP32更新

FP16训练需要GradScaler来防止梯度下溢！

📝 代码示例：FP16训练

# FP16训练
import torch
import torch.nn as nn
import torch.optim as optim

class SimpleModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(100, 200)
        self.fc2 = nn.Linear(200, 10)
    
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        return self.fc2(x)

# 初始化
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model = SimpleModel().to(device)
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 创建GradScaler
scaler = torch.amp.GradScaler(enabled=torch.cuda.is_available())

# FP16训练循环
print("\n🔢 FP16训练开始")
model.train()
for epoch in range(5):
    for _ in range(100):
        optimizer.zero_grad()
        
        inputs = torch.randn(32, 100).to(device)
        targets = torch.randint(0, 10, (32,)).to(device)
        
        # 使用autocast自动混合精度
        with torch.autocast(device_type=str(device), dtype=torch.float16):
            outputs = model(inputs)
            loss = criterion(outputs, targets)
        
        # 使用GradScaler缩放梯度
        scaler.scale(loss).backward()
        scaler.step(optimizer)
        scaler.update()
    
    print(f"Epoch {epoch+1}完成")

print("✅ FP16训练完成")

🔢 BF16训练

通俗理解：BF16训练就像用一辆越野性能好的车，可以处理各种路况（大动态范围），但速度可能不如FP16快。

优点：动态范围与FP32相同，不需要GradScaler，更容易使用

缺点：精度比FP16低，需要特定硬件支持

🎨 可视化演示：BF16 vs FP16

🔢 BF16 vs FP16对比
📊 动态范围: BF16与FP32相同，FP16较小
🎯 精度: FP16精度更高，BF16精度较低
🔧 GradScaler: FP16需要，BF16不需要
💻 硬件支持: BF16需要新硬件，FP16支持更广

BF16适合训练不稳定的场景，不需要担心梯度下溢！

✏️ 混合精度训练实战练习

通俗理解：现在我们来综合运用FP16和BF16训练，理解它们的优缺点和适用场景。

在LLM中的作用：混合精度训练是训练大型语言模型的必备技术，可以大幅提升训练效率。

📝 完整示例：混合精度训练框架

# 混合精度训练框架
import torch
import torch.nn as nn
import torch.optim as optim

class MixedPrecisionTrainer:
    def __init__(self, model, device='cuda', dtype='fp16'):
        self.model = model.to(device)
        self.device = device
        self.dtype = torch.float16 if dtype == 'fp16' else torch.bfloat16
        
        if dtype == 'fp16':
            self.scaler = torch.amp.GradScaler(enabled=torch.cuda.is_available())
        else:
            self.scaler = None
    
    def train_step(self, inputs, targets, criterion, optimizer):
        optimizer.zero_grad()
        
        with torch.autocast(device_type=self.device, dtype=self.dtype):
            outputs = self.model(inputs)
            loss = criterion(outputs, targets)
        
        if self.scaler is not None:
            self.scaler.scale(loss).backward()
            self.scaler.step(optimizer)
            self.scaler.update()
        else:
            loss.backward()
            optimizer.step()
        
        return loss.item()

print("✅ 混合精度训练器已创建")
print("支持FP16和BF16两种模式")
