📊 什么是回归损失？

通俗理解：回归损失就像老师批改作业，看你预测的分数和真实分数差多少。差得越多，损失越大。

常见类型：MSE（均方误差）、MAE（平均绝对误差）、Huber Loss（平滑L1）

🎨 可视化演示：回归损失曲线

MSE Loss
📈
误差越大，惩罚越重
MAE Loss
📐
对异常值不敏感
Huber Loss
📉
结合MSE和MAE优点

MSE对大误差惩罚很重，MAE比较稳健，Huber Loss是两者的折中！

📝 代码示例：回归损失

# 回归损失函数
import torch
import torch.nn as nn

# 模拟预测和真实值
predictions = torch.tensor([2.5, 3.2, 1.8])
targets = torch.tensor([3.0, 3.0, 2.0])

# 1. MSE Loss（均方误差）
mse_loss = nn.MSELoss()
print("MSE Loss:", mse_loss(predictions, targets).item())

# 2. MAE Loss（平均绝对误差）
mae_loss = nn.L1Loss()
print("MAE Loss:", mae_loss(predictions, targets).item())

# 3. Smooth L1 Loss（Huber Loss）
smooth_l1 = nn.SmoothL1Loss()
print("Smooth L1 Loss:", smooth_l1(predictions, targets).item())

# 4. MSELoss和L1Loss对比（有异常值时）
outlier_pred = torch.tensor([2.5, 3.2, 100.0])
outlier_target = torch.tensor([3.0, 3.0, 2.0])
print("\n有异常值时:")
print("MSE Loss:", mse_loss(outlier_pred, outlier_target).item())
print("MAE Loss:", mae_loss(outlier_pred, outlier_target).item())

🏷️ 什么是分类损失？

通俗理解：分类损失就像猜选择题，看你猜对的概率有多高。概率越高，损失越小。

常见类型：CrossEntropyLoss（交叉熵）、BinaryCrossEntropy（二元交叉熵）、FocalLoss（焦点损失）

🎨 可视化演示：分类损失对比

🏷️ 分类损失对比
📊 CrossEntropyLoss: 多分类任务的标准损失
🔘 BCELoss: 二元分类，需要sigmoid输出
🔑 BCEWithLogitsLoss: BCELoss + sigmoid（数值更稳定）
🎯 FocalLoss: 关注难分类样本

CrossEntropyLoss内部已经包含了softmax，不需要额外添加！

📝 代码示例：分类损失

# 分类损失函数
import torch
import torch.nn as nn

# 模拟多分类
logits = torch.tensor([[2.0, 1.0, 0.1], [0.5, 3.0, 0.2]])
targets = torch.tensor([0, 1])

# 1. CrossEntropyLoss（多分类）
ce_loss = nn.CrossEntropyLoss()
print("CrossEntropyLoss:", ce_loss(logits, targets).item())

# 2. NLLLoss（负对数似然，需要log_softmax输出）
log_probs = torch.log_softmax(logits, dim=1)
nll_loss = nn.NLLLoss()
print("NLLLoss:", nll_loss(log_probs, targets).item())

# 模拟二分类
binary_logits = torch.tensor([1.5, -0.5, 2.0])
binary_targets = torch.tensor([1.0, 0.0, 1.0])

# 3. BCEWithLogitsLoss（二分类，内部含sigmoid）
bce_logits = nn.BCEWithLogitsLoss()
print("\nBCEWithLogitsLoss:", bce_logits(binary_logits, binary_targets).item())

# 4. BCELoss（需要手动sigmoid）
sigmoid_output = torch.sigmoid(binary_logits)
bce_loss = nn.BCELoss()
print("BCELoss:", bce_loss(sigmoid_output, binary_targets).item())

🎯 什么是自定义损失？

通俗理解：自定义损失就像根据你的特殊需求，自己制定一套评分规则。

适用场景：标准损失函数无法满足特定任务需求时，比如不平衡数据、多任务学习等。

🎨 可视化演示：自定义损失的应用场景

🎯 自定义损失场景
⚖️ 类别不平衡: 给少数类更高权重
🎯 Focal Loss: 降低简单样本权重
🔄 多任务损失: 加权组合多个任务损失
📊 对比学习损失: 拉近正样本，推开负样本

自定义损失需要继承nn.Module或直接实现可微函数！

📝 代码示例：自定义损失

# 自定义损失函数
import torch
import torch.nn as nn

# 1. Focal Loss（焦点损失）
class FocalLoss(nn.Module):
    def __init__(self, alpha=0.25, gamma=2.0):
        super().__init__()
        self.alpha = alpha
        self.gamma = gamma
    
    def forward(self, logits, targets):
        ce_loss = nn.CrossEntropyLoss(reduction='none')(logits, targets)
        pt = torch.exp(-ce_loss)
        focal_loss = self.alpha * (pt ** self.gamma) * ce_loss
        return focal_loss.mean()

# 2. 多任务损失
class MultiTaskLoss(nn.Module):
    def __init__(self, weights=[1.0, 0.5]):
        super().__init__()
        self.weights = weights
    
    def forward(self, outputs, targets):
        loss1 = nn.CrossEntropyLoss()(outputs[0], targets[0])
        loss2 = nn.MSELoss()(outputs[1], targets[1])
        return self.weights[0] * loss1 + self.weights[1] * loss2

# 3. 对比学习损失（NT-Xent）
class NTXentLoss(nn.Module):
    def __init__(self, temperature=0.1):
        super().__init__()
        self.temperature = temperature
    
    def forward(self, z1, z2):
        batch_size = z1.size(0)
        z = torch.cat([z1, z2], dim=0)
        
        sim = torch.matmul(z, z.T) / self.temperature
        sim = sim - torch.eye(2*batch_size, device=z.device) * 1e9
        
        labels = torch.arange(batch_size, device=z.device)
        labels = torch.cat([labels, labels], dim=0)
        
        return nn.CrossEntropyLoss()(sim, labels)

✏️ 损失函数实战练习

通俗理解：现在我们来综合运用各种损失函数，理解它们在实际训练中的作用。

在LLM中的作用：损失函数是训练的目标，选择合适的损失函数对模型性能至关重要。

📝 完整示例：损失函数选择指南

# 损失函数选择指南
import torch
import torch.nn as nn

# 任务：回归预测房价
print("📊 回归任务（房价预测）:")
price_pred = torch.tensor([2.5, 3.2, 1.8, 50.0])  # 含异常值
price_target = torch.tensor([3.0, 3.0, 2.0, 4.0])

mse = nn.MSELoss()
mae = nn.L1Loss()
smooth_l1 = nn.SmoothL1Loss()

print("  MSE:", mse(price_pred, price_target).item())
print("  MAE:", mae(price_pred, price_target).item())
print("  Smooth L1:", smooth_l1(price_pred, price_target).item())

# 任务：分类识别图片
print("\n🏷️ 分类任务（图片识别）:")
image_logits = torch.tensor([[2.0, 1.0, 0.1]])
image_target = torch.tensor([0])

ce = nn.CrossEntropyLoss()
print("  CrossEntropy:", ce(image_logits, image_target).item())

# 任务：情感分析（二分类）
print("\n🔘 二分类任务（情感分析）:")
sentiment_logits = torch.tensor([1.5, -0.5])
sentiment_target = torch.tensor([1.0, 0.0])

bce_logits = nn.BCEWithLogitsLoss()
print("  BCEWithLogits:", bce_logits(sentiment_logits, sentiment_target).item())

📉 损失函数对比表

损失函数适用任务特点对异常值敏感PyTorch类
MSE回归误差平方，梯度大是nn.MSELoss
MAE回归绝对误差，稳健否nn.L1Loss
SmoothL1回归结合MSE和MAE否nn.SmoothL1Loss
CrossEntropy多分类内部含softmax-nn.CrossEntropyLoss
BCEWithLogits二分类内部含sigmoid-nn.BCEWithLogitsLoss
FocalLoss不平衡分类关注难分类样本-自定义
