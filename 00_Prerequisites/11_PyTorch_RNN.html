🔹 什么是RNN？

通俗理解：RNN就像一个有记忆的神经网络，它会记住之前看到的信息，然后用这些信息来处理当前的输入。

在LLM中的作用：虽然现在LLM主要用Transformer，但RNN是理解序列建模的基础。

hₜ = f(W·xₜ + U·hₜ₋₁ + b)

RNN的隐藏状态hₜ同时依赖当前输入xₜ和上一步的隐藏状态hₜ₋₁！

📝 代码示例

# RNN基础用法
import torch
import torch.nn as nn

# 创建RNN层
# input_size: 输入特征维度
# hidden_size: 隐藏状态维度
# num_layers: RNN层数
rnn = nn.RNN(input_size=10, hidden_size=20, num_layers=1, batch_first=True)

# 模拟输入 (batch, seq_len, input_size)
x = torch.randn(3, 5, 10)

# 前向传播
output, h_n = rnn(x)

print("输出形状：", output.shape)    # (batch, seq_len, hidden_size)
print("最后隐藏状态：", h_n.shape)  # (num_layers, batch, hidden_size)

🧠 什么是LSTM？

通俗理解：LSTM就像一个有门的记忆系统，可以选择性地记住或忘记信息，解决了RNN的梯度消失问题。

在LLM中的作用：Transformer出现之前，LSTM是NLP任务的主流模型。

🎨 可视化演示：LSTM结构

🧠 LSTM门控机制
🚪 遗忘门: 决定丢弃哪些旧记忆
➕ 输入门: 决定保存哪些新信息
📝 细胞状态: 长期记忆通道
🚪 输出门: 决定输出哪些信息

LSTM通过门控机制可以记住长距离依赖，解决了RNN的梯度消失问题！

📝 代码示例

# LSTM用法
import torch
import torch.nn as nn

# 创建LSTM层
lstm = nn.LSTM(input_size=10, hidden_size=20, num_layers=1, batch_first=True)

# 模拟输入
x = torch.randn(3, 5, 10)

# 前向传播，返回输出和(h_n, c_n)
output, (h_n, c_n) = lstm(x)

print("输出形状：", output.shape)  # (batch, seq_len, hidden_size)
print("隐藏状态h：", h_n.shape)      # (num_layers, batch, hidden_size)
print("细胞状态c：", c_n.shape)      # (num_layers, batch, hidden_size)

⚡ 什么是GRU？

通俗理解：GRU是LSTM的简化版，把遗忘门和输入门合并成更新门，参数更少，训练更快。

在LLM中的作用：作为LSTM的轻量级替代方案，常用于资源受限的场景。

🎨 可视化演示：GRU vs LSTM

⚡ GRU门控机制
🔄 更新门: 控制新信息的加入量
🔍 重置门: 控制对过去信息的遗忘程度
💡 新候选状态: 结合当前输入和重置后的过去

GRU比LSTM参数少约三分之一，训练速度更快！

📝 代码示例

# GRU用法
import torch
import torch.nn as nn

# 创建GRU层
gru = nn.GRU(input_size=10, hidden_size=20, num_layers=1, batch_first=True)

# 模拟输入
x = torch.randn(3, 5, 10)

# 前向传播
output, h_n = gru(x)

print("输出形状：", output.shape)    # (batch, seq_len, hidden_size)
print("最后隐藏状态：", h_n.shape)  # (num_layers, batch, hidden_size)

# 对比LSTM和GRU的参数数量
lstm = nn.LSTM(10, 20)
gru = nn.GRU(10, 20)
print("LSTM参数：", sum(p.numel() for p in lstm.parameters()))
print("GRU参数：", sum(p.numel() for p in gru.parameters()))

✏️ RNN实战练习

通俗理解：现在我们来构建一个RNN模型进行文本分类。

在LLM中的作用：理解序列建模的基本流程，为学习Transformer打下基础。

📝 完整示例：文本分类

# RNN文本分类
import torch
import torch.nn as nn

class TextClassifier(nn.Module):
    def __init__(self, vocab_size, embed_dim, hidden_dim, num_classes):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, embed_dim)
        self.lstm = nn.LSTM(embed_dim, hidden_dim, batch_first=True)
        self.fc = nn.Linear(hidden_dim, num_classes)

    def forward(self, x):
        x = self.embedding(x)
        _, (h_n, _) = self.lstm(x)
        return self.fc(h_n[-1])

# 创建模型
model = TextClassifier(vocab_size=10000, embed_dim=128, hidden_dim=256, num_classes=2)

# 模拟输入
input_ids = torch.randint(0, 10000, (32, 50))
output = model(input_ids)

print("输入形状：", input_ids.shape)
print("输出形状：", output.shape)
