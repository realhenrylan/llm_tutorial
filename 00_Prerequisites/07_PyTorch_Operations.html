➕ 基础运算是什么？

通俗理解：基础运算就像做加减乘除，对张量中的每个元素进行操作。

在LLM中的作用：用来做激活函数、归一化、元素级变换等。

🎨 可视化演示：元素级运算

📊 元素级运算示例
加法 +
减法 -
乘法 *
除法 /
每个元素独立运算，形状必须相同或可广播！

元素级运算使用标准算术运算符，也可以使用torch.add()、torch.mul()等函数！

📝 代码示例

# 基础元素级运算
import torch

a = torch.tensor([[1, 2], [3, 4]])
b = torch.tensor([[10, 20], [30, 40]])

print("加法：", a + b)
print("乘法：", a * b)
print("幂运算：", a ** 2)
print("取最大值：", torch.max(a, b))

🔗 矩阵运算是什么？

通俗理解：矩阵运算就像矩阵乘法，不是简单的元素相乘，而是按行和列进行点积运算。

在LLM中的作用：用来做线性变换、注意力计算、层与层之间的连接等。

矩阵乘法要求第一个矩阵的列数等于第二个矩阵的行数！

📝 代码示例

# 矩阵运算
import torch

# 矩阵乘法
A = torch.tensor([[1, 2, 3], [4, 5, 6]])
B = torch.tensor([[7, 8], [9, 10], [11, 12]])

print("矩阵乘法 @：", A @ B)
print("torch.mm：", torch.mm(A, B))

# 批量矩阵乘法
batch_A = torch.randn(2, 3, 4)
batch_B = torch.randn(2, 4, 5)
batch_C = torch.bmm(batch_A, batch_B)
print("批量矩阵乘法形状：", batch_C.shape)

🔄 维度变形是什么？

通俗理解：维度变形就像把积木重新排列，改变形状但不改变数据。

在LLM中的作用：用来调整张量形状，适配不同层的输入输出要求。

reshape前后总元素数必须相同！可以用-1让PyTorch自动计算维度。

📝 代码示例

# 维度变形
import torch

x = torch.randn(2, 3, 4)
print("原始形状：", x.shape)

# reshape
y = x.reshape(6, 4)
print("reshape(6, 4)：", y.shape)

# 使用-1自动计算
z = x.reshape(-1, 3)
print("reshape(-1, 3)：", z.shape)

# view（与reshape类似）
w = x.view(2, 12)
print("view(2, 12)：", w.shape)

📐 维度操作是什么？

通俗理解：维度操作就像在特定方向上进行汇总、拼接或拆分。

在LLM中的作用：用来做池化、注意力机制中的维度变换、batch处理等。

🎨 可视化演示：常用维度操作

🔢
sum(dim=1)
按列求和
🔢
mean(dim=0)
按行求平均
🔗
cat(dim=0)
纵向拼接
🔗
stack(dim=1)
新增维度拼接

dim=0是行方向，dim=1是列方向，dim=2是深度方向！

📝 代码示例

# 维度操作
import torch

x = torch.tensor([[1, 2, 3], [4, 5, 6]])

print("按列求和：", x.sum(dim=1))
print("按行求平均：", x.mean(dim=0))
print("按列求最大值：", x.max(dim=1))

# 拼接
a = torch.tensor([[1, 2], [3, 4]])
b = torch.tensor([[5, 6], [7, 8]])

print("cat(dim=0)：", torch.cat([a, b], dim=0).shape)
print("cat(dim=1)：", torch.cat([a, b], dim=1).shape)
print("stack(dim=0)：", torch.stack([a, b], dim=0).shape)
