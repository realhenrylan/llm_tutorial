📦 NumPy数组是什么？

通俗理解：NumPy数组就像一个整齐的格子柜，每个格子里放一个数字，按行和列排列。

在LLM中的作用：用来存储和处理大规模数据，比如词向量、权重矩阵、特征图等。

🎨 可视化演示：数组就像格子柜

(0,0)1
(0,1)2
(0,2)3
(1,0)4
(1,1)5
(1,2)6
shape = (2, 3) → 2行3列

每个格子都有坐标(row, col)，从(0,0)开始计数！

📝 代码示例

# 创建NumPy数组
import numpy as np

# 创建二维数组
arr = np.array([[1, 2, 3], [4, 5, 6]])

print("数组：", arr)
print("形状：", arr.shape)  # (2, 3)
print("访问(0,1)：", arr[0, 1])  # 2

📐 Shape与维度是什么？

通俗理解：Shape就像描述格子柜大小的标签，告诉我们有几层、每层几行几列。

在LLM中的作用：理解Tensor的形状是理解模型输入输出的关键，比如(batch, seq_len, hidden_size)。

📢 广播机制是什么？

通俗理解：广播就像自动复制小的数组，让它能和大的数组一起运算，就像给所有人发同样的消息。

在LLM中的作用：用来做批量运算，比如给一批数据加上同一个偏置，或者做element-wise操作。

小的数组会自动"广播"成大数组的形状，然后对应元素相加！

📝 代码示例

# 广播示例
import numpy as np

# (2, 3) + (1, 3) → (2, 3)
arr = np.array([[1, 2, 3], [4, 5, 6]])
bias = np.array([10, 20, 30])
result = arr + bias
print("广播结果：", result)

# (4, 1) * (1, 3) → (4, 3)
weights = np.array([[0.5], [0.6], [0.7], [0.8]])
features = np.array([[1, 2, 3]])
output = weights * features
print("输出形状：", output.shape)

🔢 Einsum表达是什么？

通俗理解：Einsum就像一种统一的"数组运算语言"，用字母表示维度，告诉计算机怎么求和、相乘和收缩维度。

在LLM中的作用：用来表达复杂的张量运算，比如矩阵乘法、Attention机制等，代码更简洁清晰。

🎨 可视化演示：Einsum的字母魔法

A[ab] × B[bc] → C[ac]
A[0,0]
A[0,1]

重复的字母b会被求和收缩掉！Einsum会自动处理维度匹配和求和。

📝 代码示例

# Einsum示例
import numpy as np

# 矩阵乘法: A[ab] @ B[bc] -> C[ac]
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
C = np.einsum('ab,bc->ac', A, B)
print("矩阵乘法：", C)

# 点积: v[a] @ w[a] -> scalar
v = np.array([1, 2, 3])
w = np.array([4, 5, 6])
dot = np.einsum('a,a->', v, w)
print("点积：", dot)
