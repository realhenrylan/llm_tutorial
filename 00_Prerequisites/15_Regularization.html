⚖️ 什么是L1/L2正则化？

通俗理解：正则化就像给模型施加"压力"，防止它变得"太胖"（参数过大）。就像老师不让学生死记硬背，而是要理解原理。

L1正则化：让一些参数变为0，相当于特征选择

L2正则化：让参数变小但不为0，防止过拟合

🎨 可视化演示：正则化效果

📐
无正则化
参数可以很大
🟢
L1正则化
部分参数为0
🔵
L2正则化
参数变小

L1像一把刀，砍掉不重要的参数；L2像一个气球，把参数都压小！

📝 代码示例：L1/L2正则化

# L1/L2正则化
import torch
import torch.nn as nn
import torch.optim as optim

# 定义模型
class RegularizedModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(10, 100)
        self.fc2 = nn.Linear(100, 1)
    
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        return self.fc2(x)

# 方法1：在优化器中使用weight_decay（L2正则化）
model1 = RegularizedModel()
optimizer1 = optim.Adam(model1.parameters(), lr=0.001, weight_decay=0.01)
print("✅ 使用weight_decay实现L2正则化")

# 方法2：手动计算L1正则化
model2 = RegularizedModel()
optimizer2 = optim.Adam(model2.parameters(), lr=0.001)

def compute_l1_regularization(model, lambda_l1=0.01):
    l1_loss = 0.0
    for param in model.parameters():
        l1_loss += torch.norm(param, 1)
    return lambda_l1 * l1_loss

print("✅ 手动计算L1正则化")

# 方法3：使用L1正则化优化器
print("✅ 使用torch.optim.SGD + L1正则化")

🎲 什么是Dropout？

通俗理解：Dropout就像上课随机让一些学生请假，这样老师就不能依赖某些学生，必须教会所有学生。模型学会更鲁棒的特征表示。

工作原理：训练时随机丢弃一部分神经元（设为0），测试时恢复所有神经元但乘以概率

🎨 可视化演示：Dropout效果

🎲 Dropout机制
🔄 训练阶段: 随机丢弃50%的神经元
✅ 测试阶段: 使用全部神经元，但输出乘以0.5
🎯 效果: 防止神经元之间的过度依赖

训练时model.train()会启用Dropout，测试时model.eval()会禁用Dropout！

📝 代码示例：Dropout

# Dropout正则化
import torch
import torch.nn as nn

# 带Dropout的模型
class DropoutModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(10, 100)
        self.dropout = nn.Dropout(p=0.5)  # 丢弃50%
        self.fc2 = nn.Linear(100, 1)
    
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.dropout(x)  # 应用Dropout
        return self.fc2(x)

model = DropoutModel()

# 训练模式（启用Dropout）
model.train()
x = torch.randn(1, 10)
output_train = model(x)
print("训练模式输出:", output_train)
print("训练模式fc1输出中非零比例:", (model.fc1(x) != 0).float().mean().item())

# 评估模式（禁用Dropout）
model.eval()
output_eval = model(x)
print("\n评估模式输出:", output_eval)

# Dropout也可以用于特定层
class MultiDropoutModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(10, 100)
        self.dropout1 = nn.Dropout(p=0.3)
        self.fc2 = nn.Linear(100, 50)
        self.dropout2 = nn.Dropout(p=0.5)
        self.fc3 = nn.Linear(50, 1)
    
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.dropout1(x)
        x = torch.relu(self.fc2(x))
        x = self.dropout2(x)
        return self.fc3(x)

🚀 什么是优化器？

通俗理解：优化器就像导航系统，帮助模型找到损失函数的最低点。不同的优化器有不同的导航策略。

常见优化器：SGD、Adam、AdamW、RMSprop、Adagrad

🎨 可视化演示：优化器对比

🚀 优化器对比
🏃 SGD: 简单直接，动量版效果更好
💨 Momentum: 像小球下坡，积累速度
⚡ Adam: 自适应学习率，最常用
🔧 AdamW: Adam + 权重衰减解耦，更稳定

AdamW是Adam的改进版，在训练大型模型时更稳定！

📝 代码示例：各种优化器

# 各种优化器
import torch
import torch.nn as nn
import torch.optim as optim

# 定义模型
model = nn.Linear(10, 1)

# 1. SGD
opt_sgd = optim.SGD(model.parameters(), lr=0.01)

# 2. SGD + Momentum
opt_momentum = optim.SGD(model.parameters(), lr=0.01, momentum=0.9)

# 3. Adam（最常用）
opt_adam = optim.Adam(model.parameters(), lr=0.001)

# 4. AdamW（推荐用于大型模型）
opt_adamw = optim.AdamW(model.parameters(), lr=0.001, weight_decay=0.01)

# 5. RMSprop
opt_rmsprop = optim.RMSprop(model.parameters(), lr=0.001)

# 6. Adagrad
opt_adagrad = optim.Adagrad(model.parameters(), lr=0.01)

print("✅ 各种优化器已初始化")

# 优化器使用示例
x = torch.randn(32, 10)
y = torch.randn(32, 1)

for epoch in range(10):
    opt_adam.zero_grad()
    output = model(x)
    loss = nn.MSELoss()(output, y)
    loss.backward()
    opt_adam.step()

print("✅ Adam优化器训练完成")

✏️ 正则化与优化实战练习

通俗理解：现在我们来综合运用正则化和优化器，构建一个更稳健的模型。

在LLM中的作用：正则化和优化器是训练稳定模型的关键，特别是在训练大型语言模型时。

📝 完整示例：综合正则化策略

# 综合正则化策略
import torch
import torch.nn as nn
import torch.optim as optim

class RegularizedModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(20, 200)
        self.bn1 = nn.BatchNorm1d(200)
        self.dropout1 = nn.Dropout(p=0.3)
        
        self.fc2 = nn.Linear(200, 100)
        self.bn2 = nn.BatchNorm1d(100)
        self.dropout2 = nn.Dropout(p=0.5)
        
        self.fc3 = nn.Linear(100, 10)
    
    def forward(self, x):
        x = torch.relu(self.bn1(self.fc1(x)))
        x = self.dropout1(x)
        
        x = torch.relu(self.bn2(self.fc2(x)))
        x = self.dropout2(x)
        
        return self.fc3(x)

# 初始化
model = RegularizedModel()
criterion = nn.CrossEntropyLoss()
optimizer = optim.AdamW(model.parameters(), lr=0.001, weight_decay=0.01)

print("✅ 综合正则化模型已准备就绪")
print("策略: BatchNorm + Dropout + AdamW + Weight Decay")
