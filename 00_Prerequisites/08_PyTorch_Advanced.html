🧩 nn.Module是什么？

通俗理解：nn.Module就像一个积木盒子，可以把各种层(Layer)组合在一起，形成一个完整的模型。

在LLM中的作用：用来构建神经网络模型，管理参数、前向传播和设备迁移。

🎨 可视化演示：Module的结构

🧩 MyModel (nn.Module)
• Linear(in_features=10, out_features=20)
• ReLU()
• Linear(in_features=20, out_features=5)

所有PyTorch模型都继承自nn.Module，需要实现forward()方法！

📝 代码示例

# 定义模型
import torch
import torch.nn as nn

class SimpleModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super().__init__()
        self.fc1 = nn.Linear(input_size, hidden_size)
        self.relu = nn.ReLU()
        self.fc2 = nn.Linear(hidden_size, output_size)

    def forward(self, x):
        x = self.fc1(x)
        x = self.relu(x)
        x = self.fc2(x)
        return x

# 创建模型
model = SimpleModel(input_size=10, hidden_size=20, output_size=5)
print("模型结构：", model)

⚙️ Optimizer是什么？

通俗理解：Optimizer就像一个调节器，根据梯度来调整模型参数，让模型不断改进。

在LLM中的作用：用来更新模型权重，常用Adam和AdamW。

🎨 可视化演示：优化器的工作流程

⚙️ 优化器工作流程
1️⃣ 计算损失 → loss
2️⃣ 反向传播 → loss.backward()
3️⃣ 优化器更新 → optimizer.step()
4️⃣ 清除梯度 → optimizer.zero_grad()

训练时必须先调用zero_grad()清除梯度，再计算损失和反向传播！

📝 代码示例

# 使用优化器
import torch
import torch.nn as nn

class SimpleModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.linear = nn.Linear(10, 5)

    def forward(self, x):
        return self.linear(x)

model = SimpleModel()

# 创建优化器
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)

# 训练步骤
optimizer.zero_grad()
output = model(torch.randn(2, 10))
loss = output.sum()
loss.backward()
optimizer.step()

print("优化器类型：", optimizer.__class__.__name__)
print("学习率：", optimizer.param_groups[0]['lr'])

🔄 训练循环是什么？

通俗理解：训练循环就像一个生产线，不断重复"输入数据→计算损失→更新参数"的过程。

在LLM中的作用：用来训练模型，让模型从数据中学习规律。

🎨 可视化演示：训练循环流程

🔄 训练循环流程图
📥 输入数据 (batch)
→
🧠 前向传播 (forward)
→
📊 计算损失 (loss)
→
🔙 反向传播 (backward)
→
⚙️ 更新参数 (step)

训练时模型要设为train()模式，评估时要设为eval()模式！

📝 代码示例

# 完整训练循环
import torch
import torch.nn as nn

model = nn.Linear(10, 5)
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
criterion = nn.MSELoss()

# 训练模式
model.train()

# 训练循环
for epoch in range(10):
    # 清除梯度
    optimizer.zero_grad()
    
    # 模拟数据
    x = torch.randn(32, 10)
    y = torch.randn(32, 5)
    
    # 前向传播
    output = model(x)
    
    # 计算损失
    loss = criterion(output, y)
    
    # 反向传播
    loss.backward()
    
    # 更新参数
    optimizer.step()
    
    print(f"Epoch {epoch}, Loss: {loss.item():.4f}")

💾 模型保存与加载是什么？

通俗理解：模型保存就像把训练好的模型"存档"，方便以后继续训练或使用。

在LLM中的作用：保存训练好的模型权重，支持断点续训和推理部署。

🎨 可视化演示：保存与加载

💾 模型保存方式
• torch.save(model, 'model.pt') — 保存完整模型
• torch.save(model.state_dict(), 'weights.pt') — 仅保存权重
• torch.load() — 加载模型

推荐只保存state_dict()，更灵活且文件更小！

📝 代码示例

# 模型保存与加载
import torch
import torch.nn as nn

class SimpleModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.linear = nn.Linear(10, 5)

    def forward(self, x):
        return self.linear(x)

model = SimpleModel()

# 保存权重（推荐）
torch.save(model.state_dict(), 'model_weights.pt')

# 加载权重
new_model = SimpleModel()
new_model.load_state_dict(torch.load('model_weights.pt'))
new_model.eval()

print("模型加载成功！")
