📦 什么是Tensor？

通俗理解：Tensor就像一个超级数组，可以在CPU和GPU上运算，是PyTorch的核心数据结构。

在LLM中的作用：用来存储模型参数、输入数据、中间计算结果等，是深度学习的基石。

🎨 可视化演示：Tensor就像数据积木

Tensor不仅存储数据，还记录了形状、数据类型和所在设备，方便自动求导！

📝 代码示例

# 创建基本Tensor
import torch

# 从列表创建
x = torch.tensor([[1, 2, 3], [4, 5, 6]])

print("Tensor：", x)
print("形状：", x.shape)
print("数据类型：", x.dtype)
print("设备：", x.device)

✨ 如何创建Tensor？

通俗理解：创建Tensor就像做蛋糕，可以从零开始做，也可以用现成的配料。

在LLM中的作用：初始化模型参数、创建输入数据、生成掩码等。

🎨 可视化演示：常见的创建方式

🌀
zeros()
创建全0张量
🔢
ones()
创建全1张量
🎲
randn()
创建随机张量
📐
eye()
创建单位矩阵

使用randn()创建的张量值服从标准正态分布（均值为0，方差为1）！

📝 代码示例

# 多种创建Tensor的方式
import torch

# 创建全0张量
zeros = torch.zeros(2, 3)
print("全0：", zeros)

# 创建全1张量
ones = torch.ones(3, 3)
print("全1：", ones)

# 创建随机张量（标准正态分布）
rand_tensor = torch.randn(2, 4)
print("随机：", rand_tensor)

# 创建单位矩阵
identity = torch.eye(3)
print("单位矩阵：", identity)

⚡ 设备管理是什么？

通俗理解：设备管理就像给Tensor分配工作地点，可以放在CPU上慢慢算，也可以放在GPU上加速算。

在LLM中的作用：利用GPU加速训练和推理，处理大规模数据。

🎨 可视化演示：CPU vs GPU

💻
CPU
  • • 通用性强
  • • 计算慢
  • • 适合简单运算
↔️
🚀
GPU
  • • 并行计算
  • • 计算快
  • • 适合深度学习

使用.to(device)可以在CPU和GPU之间移动Tensor！

📝 代码示例

# 设备管理
import torch

# 检查是否有GPU可用
device = 'cuda' if torch.cuda.is_available() else 'cpu'
print("当前设备：", device)

# 创建在CPU上的Tensor
x = torch.tensor([[1, 2], [3, 4]])
print("CPU上的Tensor：", x.device)

# 移动到GPU
x_gpu = x.to(device)
print("GPU上的Tensor：", x_gpu.device)

# 创建时直接指定设备
y = torch.tensor([[1, 2]], device=device)
print("直接创建在设备上：", y.device)

📊 数据类型是什么？

通俗理解：数据类型就像告诉计算机用多大的箱子装数字，箱子越大精度越高，但占空间也越大。

在LLM中的作用：选择合适的数据类型可以节省显存、加速训练，常用float32和bf16。

🎨 可视化演示：常用数据类型

🔵
float32
4字节 | 标准精度
🟢
bf16
2字节 | 脑浮点
🟡
float16
2字节 | 半精度
🔴
int64
8字节 | 整数索引

bf16是LLM训练中常用的数据类型，精度足够且节省一半显存！

📝 代码示例

# 数据类型转换
import torch

# 创建不同数据类型的Tensor
x_float32 = torch.tensor([1.0, 2.0, 3.0], dtype=torch.float32)
x_bf16 = torch.tensor([1.0, 2.0, 3.0], dtype=torch.bfloat16)
x_int = torch.tensor([1, 2, 3], dtype=torch.int64)

print("float32：", x_float32.dtype)
print("bfloat16：", x_bf16.dtype)
print("int64：", x_int.dtype)

# 类型转换
x_converted = x_float32.to(torch.bfloat16)
print("转换后：", x_converted.dtype)
