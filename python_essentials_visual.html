📋 List 列表是什么？

通俗理解：List就像一个排队的队伍，每个人（数据）都有自己的位置（索引），按顺序排列。

在LLM中的作用：用来存储顺序数据，比如训练时的批次数据、token序列等。

🎨 可视化演示：List就像排队

0 "llm"
1 "algo"
2 "leetcode"

每个元素都有对应的索引（从0开始），就像排队时的序号一样！

📝 代码示例

# 创建一个列表
tokens = ["llm", "algo", "leetcode"]

# 通过索引访问元素
print(tokens[0])  # 输出: "llm"
print(tokens[1])  # 输出: "algo"

# 添加元素
tokens.append("python")
print(tokens)  # ["llm", "algo", "leetcode", "python"]

📖 Dict 字典是什么？

通俗理解：Dict就像一本字典，每个词（键key）对应一个解释（值value），通过词能快速找到解释。

在LLM中的作用：用来存储配置信息、模型参数、统计结果等命名状态。

🎨 可视化演示：Dict就像查字典

"name"
"TinyLM"
"precision"
"bf16"
"batch_size"
4

每个键值对就像字典里的词条，通过键（key）快速找到对应的值（value）！

📝 代码示例

# 创建一个字典
config = {
    "name": "TinyLM",
    "precision": "bf16",
    "batch_size": 4
}

# 通过键访问值
print(config["name"])      # 输出: "TinyLM"
print(config["batch_size"])  # 输出: 4

# 修改值
config["batch_size"] = 8
print(config)  # {"name": "TinyLM", "precision": "bf16", "batch_size": 8}

🏗️ Class 类是什么？

通俗理解：Class就像一个模具，用来批量生产具有相同特征的对象。比如用同一个模具做饼干。

在LLM中的作用：用来封装配置、状态、结果等，让代码更整洁、可复用。

🎨 可视化演示：Class就像做饼干

🍪 饼干模具 (Class)
形状：圆形
大小：中等
方法：烘焙()
⬇️
🍪 饼干1 (实例1)
形状：圆形
大小：中等
🍪 饼干2 (实例2)
形状：圆形
大小：中等

Class是模板，实例是根据模板创建的具体对象。修改模板会影响所有实例，但每个实例可以有自己的独特属性！

📝 代码示例

from dataclasses import dataclass

# 定义一个类（模具）
@dataclass
class BatchSummary:
    loss: float
    step: int
    extras: dict
    
    # 方法：生成摘要字符串
    def short(self):
        return f"step={self.step}, loss={self.loss:.4f}"

# 创建实例（用模具做饼干）
summary = BatchSummary(loss=0.1234, step=12, extras={"lr": 1e-4})
print(summary.short())  # 输出: "step=12, loss=0.1234"

🔒 With 语句是什么？

通俗理解：With就像一个自动开关，进入时打开资源，离开时自动关闭，不用担心忘记关。

在LLM中的作用：管理文件、临时目录等资源，确保资源正确释放，避免内存泄漏。

🎨 可视化演示：With就像自动门

🔓 with 语句开始 - 打开资源

进入 with 块

↓

使用资源（如文件、临时目录）

↓

执行操作...

⬇️
🔒 with 语句结束 - 自动关闭资源

离开 with 块

↓

自动清理资源

↓

无需手动关闭！

With语句确保即使发生错误，资源也会被正确关闭。这是Python的"上下文管理器"机制！

📝 代码示例

from pathlib import Path
from tempfile import TemporaryDirectory

# 使用 with 管理临时目录
with TemporaryDirectory() as tmpdir:
    root = Path(tmpdir)
    report = root / 'report.txt'
    report.write_text('hello io', encoding='utf-8')
    print('临时文件路径：', report)
    print('临时文件内容：', report.read_text(encoding='utf-8'))

# with 块结束后，临时目录自动清理
print('✅ with 收尾完成')
