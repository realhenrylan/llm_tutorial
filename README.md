## 📝 使用说明

1. **在线预览**：点击上述表格中的"在线预览"链接即可直接在浏览器中查看 HTML 文件内容
2. **本地运行**：克隆仓库后使用 Python 启动本地服务器进行预览

```bash
git clone https://github.com/realhenrylan/llm_tutorial.git
cd llm_tutorial
python3 -m http.server 8000
```

然后访问 `http://localhost:8000` 查看。

## 📄 文件格式说明

- `.html`：HTML 格式的教程文件，可直接在浏览器中查看
- `.ipynb`：Jupyter Notebook 文件，可在 Jupyter 或 Google Colab 中打开
